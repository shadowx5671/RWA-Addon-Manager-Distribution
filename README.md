# RWA-Addon-Manager

Ein einfacher und benutzerfreundlicher Manager zum Installieren und Verwalten von Add-ons von Railworks Austria für den Train Simulator Classic.
Mit dem Railworks Manager kannst du Add-ons schnell, übersichtlich und ohne komplizierte manuelle Schritte installieren und verwalten.


# ✨ **Features**
- 📦 Einfache Installation von Railworks Austria Add-ons
- 🗑️ Deinstallation von installierten Inhalten
- 🔄 Update-Funktion für bestehende Add-ons
- 🎨 Installation von Repaints
- 🔊 Unterstützung für Sound-Updates und Erweiterungen
- 🧩 Installation von Extensions
- ⚙️ Ausführen hilfreicher Konfigurationen über Scripts
- 📂 Übersichtliche Verwaltung aller installierten Inhalte

# 🚀 **Installation**
- Lade die neueste Version von den Releases herunter
- Starte den Railworks Manager
- Wähle dein Train Simulator Classic Verzeichnis
- Melde dich mit deinen Daten an
- Beginne mit der Installation deiner Add-ons
